# LogWatch-Security-Monitor
Python-based security log monitoring tool that detects suspicious activity, assigns risk levels, and generates incident reports.

## Features
- Scans logs for suspicious activity
- Assigns severity and risk scores
- Detects brute-force login attempts
- Generates audit reports
- Writes alert history to file
- Flags suspicious IP addresses

## Technologies
- Python 3
- Regular Expressions
- File I/O
- Data Structures (Counter, Dict)

## How to Run
1. Place your log file in the project folder
2. Update LOG_FILE in the script
3. Run:

python logwatch.py

4. View output in terminal and in generated report files

## Example Use Case
Used by security teams to monitor system activity and identify
potential intrusion attempts.
